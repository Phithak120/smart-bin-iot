หากคุณตัดสินใจเลือกวิธี **Render.com** (ซึ่งผมแนะนำที่สุดสำหรับงานส่งอาจารย์) นี่คือ Roadmap 5 ขั้นตอนที่จะทำให้เว็บของคุณออนไลน์ได้จริงครับ

---

### **ขั้นตอนที่ 1: เตรียมไฟล์ในเครื่องให้พร้อม**

ตรวจสอบว่าในโฟลเดอร์โปรเจคของคุณมีไฟล์เหล่านี้ครบถ้วนและชื่อถูกต้อง:

1. `server.js` (Backend)
2. `index.html`, `history.html`, `summary.html` (Frontend)
3. `home.js`, `history.js`, `summary.js`, `style.css` (Assets)
4. `serviceAccountKey.json` (คีย์ Firebase)
5. **`package.json`** (สร้างใหม่ตามโค้ดที่ผมให้ไว้ในข้อความก่อนหน้า)

**⚠️ สำคัญมาก:** ในไฟล์ `server.js` บรรทัดที่ตั้งพอร์ต ต้องแก้เป็น:

```javascript
const port = process.env.PORT || 3000;

```

---

### **ขั้นตอนที่ 2: เอาโค้ดขึ้น GitHub (แบบฟรีและปลอดภัย)**

1. ไปที่ [GitHub](https://github.com/) แล้วสร้าง **New Repository**
2. **สำคัญ:** เลือกเป็น **Private** (เพื่อให้คนอื่นไม่เห็นคีย์ Firebase ของเรา)
3. อัปโหลดไฟล์ทั้งหมดขึ้นไป (ยกเว้นโฟลเดอร์ `node_modules` ถ้ามี)

---

### **ขั้นตอนที่ 3: เชื่อมต่อกับ Render.com**

1. ไปที่ [Render.com](https://render.com/) และสมัครใช้งาน (เลือก Login ด้วย GitHub จะง่ายที่สุด)
2. กดปุ่ม **New +** สีน้ำเงิน แล้วเลือก **Web Service**
3. มองหารายชื่อ Repository ของคุณใน GitHub แล้วกดปุ่ม **Connect**

---

### **ขั้นตอนที่ 4: ตั้งค่าการรัน (Configuration)**

เมื่อกด Connect แล้ว ให้กรอกข้อมูลดังนี้:

* **Name:** ตั้งชื่อโปรเจคของคุณ (เช่น `smart-trash-bin-project`)
* **Environment:** เลือก `Node`
* **Region:** เลือกที่ใกล้ไทยที่สุด (เช่น `Singapore`)
* **Branch:** ปกติจะเป็น `main`
* **Build Command:** `npm install`
* **Start Command:** `node server.js`
* **Instance Type:** เลือก **Free**

---

### **ขั้นตอนที่ 5: รอการติดตั้ง (Deploy)**

1. กดปุ่ม **Create Web Service** ด้านล่างสุด
2. หน้าจอจะขึ้นแถบสถานะการติดตั้ง (คล้ายๆ ใน Terminal) ให้รอจนกว่าจะขึ้นคำว่า **"Your service is live"**
3. มองหาลิงก์ที่อยู่ด้านบนซ้าย (เช่น `https://smart-trash-bin-xxx.onrender.com`)
4. ก๊อปปี้ลิงก์นั้นไปเปิดในบราวเซอร์ หรือส่งให้อาจารย์ได้เลย!

---

### **💡 ข้อควรจำหลังใช้งาน**

* **ถ้าแก้โค้ด:** แค่คุณอัปเดตไฟล์ใหม่ขึ้น GitHub ตัว Render จะรู้ตัวและทำการอัปเดตเว็บออนไลน์ให้คุณอัตโนมัติภายใน 1-2 นาทีครับ
